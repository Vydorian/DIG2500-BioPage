# Cloaked in Cosplay

## Project Description

Cloaked in Cosplay is a responsive single-page cosplay portfolio created using HTML and CSS. The website highlights our Rogue and Gambit cosplays while showcasing our costume building process and future projects.

## Purpose

The purpose of this interface is to introduce visitors to our cosplay brand and display our work in an organized, visually appealing way.

## Intended User

The intended users are cosplay fans, convention attendees, and anyone interested in our costume projects.

## Main User Goal

Visitors can learn about our cosplay process, browse photos, and visit our TikTok page to see additional content.

## Features

- Responsive layout
- Navigation menu
- Image gallery
- Hover effects on buttons
- Behind the Build section
- Future cosplay section
- External TikTok link
